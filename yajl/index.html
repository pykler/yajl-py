yajl¶

Pure Python wrapper to the Yajl C library

yajl.__version__¶

Version of yajl-py

yajl.yajl_version¶

Version of the yajl library that was loaded

yajl.check_yajl_version()¶
Return type:bool

Returns True, if the version of yajl is identical to the version of yajl-py otherwise displays a RuntimeWarning and returns False.

Sub-Modules¶

All sub-modules are conveniently imported directly under yajl, therefore using yajl.yajl_parse.YajlParser is the same as yajl.YajlParser.
